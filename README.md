# Number-Gussing-Flask-Server
A simple Flask web app where users guess a number between 0–9. The app responds with fun GIFs depending on whether the guess is too high, too low, or correct. Built using Python, Flask routing, and dynamic URLs as part of learning backend web development.
🚀 Features
Random number generated by the server
Guess the number through the URL
GIF feedback for:
Too high 📈
Too low 📉
Correct answer 🎉

Beginner-friendly Flask project
🛠️ Built With
🐍 Python
🌶️ Flask
🎞️ Giphy GIFs
HTML (inside Flask routes)
